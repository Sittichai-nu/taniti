# Taniti Tourism Website
## Low-Fidelity Wireframe & Interactive Prototype
### WGU Performance Assessment — HJN1 UX Design

## 🌴 Project Overview

This repository contains the low-fidelity wireframe and interactive prototype for the **Taniti Tourism Website**, developed as part of a UX Design performance assessment (HJN1) at Western Governors University. The Tanitian government contracted this UX design project to increase tourism by replacing their outdated single-page website with a modern, mobile-friendly, and fully navigable tourism platform.

**Live Prototype:** https://sittichai-nu.github.io/taniti/

---

## 🎯 Project Objectives

- Redesign the Taniti tourism website to improve user experience
- Organize scattered content into clearly separated, navigable pages
- Add booking links and direct connections to activities and lodging
- Ensure mobile compatibility across all pages
- Align design decisions with survey data and user persona research

---

## 📄 Pages Included

| File | Page | Description |
|---|---|---|
| index.html | Home | Hero section, 4 feature tiles, footer |
| accommodations.html | Accommodations | Filter sidebar, 6 lodging cards with Book Now buttons |
| activities.html | Activities and Attractions | Filter bar, map placeholder, 9 activity cards |
| dining.html | Dining | 10 restaurants, cuisine types, hours, grocery stores |
| transportation.html | Transportation | Arrival options, 7-row transport table |
| faq.html | Travel Tips and FAQs | 8 expandable accordion FAQ items |
| style.css | Shared Styles | CSS stylesheet used across all pages |

---

## 🔑 Key UX Design Features

- Persistent navigation bar on every page with active state highlighting
- Search bar in the header for quick content access
- Family-Friendly filter tags on activity cards based on user persona research
- Booking links on all accommodation and activity cards
- FAQ accordion for quick access to practical travel information
- Mobile-responsive layout addressing the primary problem with the original site
- Transportation quick-link tile on homepage for first-time visitors

---

## 👤 User Persona Summary

Based on survey data from 20 Taniti tourists:

- **Typical visitor:** American adult, age 40–45, middle income (~$75,000/year)
- **Travel party:** Couple or family of 3–5
- **Purpose:** Leisure vacation or family holiday
- **Stay:** 4–7 days, arriving by air
- **Accommodation:** Hotel preferred
- **Priority needs:** Family-friendly activities, clear transportation info, booking links

---

## 🧪 Guerrilla Usability Testing

Three users tested the wireframe and provided feedback. Five actionable items were identified and incorporated into the prototype:

1. Added island tagline to hero section for immediate context
2. Added Transportation tile to homepage for first-time visitors
3. Added search bar to persistent header
4. Added Family-Friendly filter tags to activity cards
5. Promoted FAQ link in mobile footer for better discoverability
